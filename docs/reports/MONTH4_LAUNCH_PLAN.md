**代理池配置**:
```typescript
// lib/scraper/proxy-pool.ts
const PROXY_PROVIDERS = [
  { name: 'cloudflare-worker', weight: 50 },
  { name: 'residential-proxy', weight: 30 },
  { name: 'direct', weight: 20 },
]

// 自动切换策略
- 连续失败 3 次 → 降低权重
- 成功率 < 50% → 暂时禁用
- 每小时重新评估权重
```

| API 响应时间 | P95 > 500ms | 待做 |
| 错误率 | > 1% | 待做 |
| 数据库连接 | 连接池耗尽 | 待做 |

**Sentry 告警配置**:
```typescript
// sentry.server.config.ts
Sentry.init({
  dsn: process.env.SENTRY_DSN,
  tracesSampleRate: 0.1,
  profilesSampleRate: 0.1,

  // 自定义告警规则
  beforeSend(event) {
    // 过滤已知错误
    if (event.exception?.values?.[0]?.type === 'RateLimitError') {
      return null
    }
    return event
  },
})
```
