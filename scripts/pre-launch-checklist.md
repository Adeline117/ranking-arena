### 已修复的 UI 问题 🔧 (2026-01-25)
2. **关注按钮不显示"取消关注"**
   - 原因: `UserFollowButton.tsx` 和 `TraderFollowButton.tsx` 发送 GET/POST 请求时未包含 Authorization header
   - API `/api/users/follow` 和 `/api/follow` 需要 Authorization header 来验证用户身份
   - 修复: 使用 `useAuthSession` hook 的 `getAuthHeadersAsync()` 方法获取认证头，并在所有请求中包含
